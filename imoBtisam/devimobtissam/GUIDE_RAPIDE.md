## 🎯 COMMENT UTILISER

### 1. Démarrer
- Ouvrez `index.html` dans votre navigateur
- Ou testez avec `test.html` d'abord

### 2. Ajouter une Propriété
```javascript
// Dans js/script.js, ajoutez :
{
    id: 7,
    type: 'villa',
    title: 'Nouvelle Villa',
    price: '5,000,000 MAD',
    image: 'images/votre-image.jpg',
    // ...
}
```

### 3. Modifier les Couleurs
```css
/* Dans css/styles.css */
:root {
    --primary: #b8860b;     /* Votre couleur */
    --morocco-gold: #d4af37;
}
```

### 4. Changer les Textes
- Contact : Modifiez dans `index.html`
- Configuration : Modifiez dans `js/config.js`
